### There are different types of shell

1. bash
2. sh
3. ksh
4. tsh
5. fish
6. zsh

- check which shell i am using

```bash
echo $0 -> -zsh
```

```bash
echo $SHELL -> /bin/zsh
```

- check how many shells have in my system

```bash
cat /etc/shells
```

> Run using

```bash
./script.sh
bash script.sh
```
